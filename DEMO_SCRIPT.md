# Quick Demo Script - Literature Survey System

## 🚀 5-Minute Quick Demo

### Setup (Before Demo)
```bash
# Terminal 1 - Backend
cd packages/backend
npm run dev

# Terminal 2 - Frontend  
cd packages/frontend
npm run dev

# Open browser: http://localhost:5174
```

---

## 📝 Demo Flow

### 1. Home Page (30 seconds)
**Say:** "This is our Literature Survey Generation System. It uses multiple AI models to automatically create comprehensive literature reviews."

**Show:**
- Clean, modern interface
- Search input field
- Feature cards (Lightning Fast, Quality Assured, Export Ready)
- Stats section (15+ papers, 5 stages, 3 formats)

---

### 2. Create Survey (30 seconds)
**Say:** "Let me demonstrate by generating a survey on machine learning in healthcare."

**Do:**
- Type: "Machine learning applications in medical imaging"
- Click "Generate Survey"

**Say:** "The system now triggers our 5-stage pipeline..."

---

### 3. Progress Tracking (1 minute)
**Say:** "Watch as the system progresses through each stage:"

**Show each stage:**
1. 🔍 **Query Expansion** - "Generating comprehensive search queries"
2. 📄 **Paper Retrieval** - "Fetching papers from scholarly databases"
3. ✓ **Validation** - "Validating paper quality and relevance"
4. ⭐ **Evaluation** - "Ranking papers by citation count and importance"
5. 📝 **Synthesis** - "Creating coherent literature survey"

**Say:** "Each stage uses specialized AI models for optimal results."

---

### 4. Survey Results (2 minutes)
**Say:** "Here's our generated survey!"

**Show:**
- **Header:** Topic, metadata (15 papers, 850 words, date)
- **Export buttons:** PDF, DOCX, JSON
- **Research Papers Section:** Beautiful cards showing all papers
- **Survey Content:**
  - Introduction
  - Current Research Trends
  - Methodological Approaches
  - Future Directions
  - Conclusion
  - References

**Say:** "Notice the professional formatting, proper citations, and organized structure."

---

### 5. Export Demo (30 seconds)
**Say:** "Researchers can export in multiple formats."

**Do:**
- Click "Export JSON" button
- Show the downloaded file

**Say:** "PDF and DOCX are also available for direct use in research papers."

---

### 6. Additional Features (30 seconds)
**Say:** "The system also includes:"

**Show:**
- Dark mode toggle (click it)
- About page (navigate)
- Documentation page

**Say:** "Everything is documented and ready for production use."
